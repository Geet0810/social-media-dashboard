# Social Media Engagement Dashboard

An interactive dashboard for analyzing social media engagement metrics with multiple visualization types and advanced analytics.

## Features

1. **Pareto Chart** - Platform contribution analysis
2. **Scatter Plot** - Correlation analysis between metrics
3. **Historical Prediction Chart** - Time-based forecasting
4. **Dual Axis Chart** - Multi-metric comparison
5. **Outlier Detection** - Anomaly detection visualization
6. **Waterfall Chart** - Performance decomposition
7. **Comparison Matrix** - Platform/post-type comparison
8. **Margin Profitability** - Engagement efficiency
9. **Donut Chart** - Engagement distribution
10. **Growth Trends** - Time-series analysis
11. **Cumulative Performance Tracker** - Running totals
12. **Apriori Algorithm** - Association rule mining
13. **ROC Curve** - High Value Customer identification
14. **Performance Matrix** - Multi-dimensional analysis
15. **What-If Analysis** - Scenario simulation
